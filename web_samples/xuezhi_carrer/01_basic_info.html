网络工程师

  • 53
图集（15张）

网络工程师的工作内容：

1. 负责机房内的网络连接及网络间的系统配置。

2. 负责系统网络的拓扑图的建立和完善，并做好系统路由的解析和资料的整理。

3. 负责机房线路的布置和协议的规范工作。

4. 负责计算机间的网络连接及网络共享，并负责网络间安全性的设置。

5. 负责对网络障碍的分析，及时处理和解决网络中出现的问题。

6. 利用网络测试分析仪，定期对现有的网络进行优化工作。

7. 负责网络平台框架的布局和设置；如java软件工程师和java网络工程师。

8. 负责网络平台信息的采集和录入支持；如：信息技术工程师。

9. 负责网络平台的推广方向和推广模式，如：网络推广大师。

10. 负责网络平台的运作方向以及平台维护管理等工作，如：网络运营工程师。

11. 负责网络平台发展到一定阶段的商业模式和盈利方向；如：网站商务工程师，电子商务工程师。

12. 负责网络产品的定位和封装；如：项目工程师。

1职业定义

职业定义：网络工程师是通过学习和训练,掌握网络技术的理论知识和操作技能的网络技术人员。网络工程师能够从事计算机信息系统的设计、建设、运行和维护工作。

相关职业：系统集成工程师、计算机硬件工程师、

职业其他名称：网络管理员、软件工程师

点击了解：月薪5千—3万？计算机类专业薪酬跨度竟然这么大

2任务职责

工作内容、任务、职责：

1. 维护日常网络安全管理，预防网络安全隐患，及时处理网络问题。

2. 通过运用各种安全产品和技术进行安全制度建设、安全技术规划、日常维护管理、信息安全检查、审计系统账号管理及系统日志检查等。

3. 负责网络平台框架的布局和设置、网络连接、网络间的系统配置和协议规范。

4. 负责系统网络的拓扑图的建立和完善。

图集（4张）

工作考核要求：

1. 软件设计模式，系统机能需求分析和设计合理性。

2. 项目完成情况，含网络架构、系统开发的数量、性能、用时、成本等。

3. 考察软件环境运行稳定与健壮性。

4. 考察编写的代码复用性、可扩展性、易读性。

5. 考察软件故障处理情况，用户反馈情况。

3工作环境

工作场所：

通常都在写字楼内,工作场所包括办公室和机房。
图集（5张）

环境具体描述：

1.通常为隔间式办公室,每个人都有独立的工作空间。
2.办公室配备电脑和其他所需办公用品。
3.机房一般干净明亮、温度适中。
4.公司还可能设有会议室、洽谈室、休闲厅、餐厅和文体活动室等。

4知识背景

学历要求：

大学本科及以上

行业知识背景：

1. 了解计算机/互联网和电子行业的发展、趋势。

2. 对互联网市场有一定的预判能力。

3. 了解行业前沿资讯,迅速理解新体系新架构。

4. 熟悉常见通信协议,了解网络上消息交互的基本流程。

专业知识背景：

1. ORACLE/DB2数据库、Linux/UNIX系统、人机交互等。

2. C语言/JAVA/面向对象编程、数据结构。

3. 计算机与网络知识、TCP/IP等网络协议、安全协议、网络安全的基本技术。

4. 量化计算、云计算与服务、大数据的挖掘、虚拟化、软件容灾与安全等。

专业要求：

计算机类

5大学课程

重要的大学课程：

计算机网络 计算机原理 数据结构 数据库 通信原理 计算机操作系统

重要未开设的大学课程：

网络协议 网络安全体系结构 系统架构 思维导图 网络系统设计与管理 服务器管理与维护

3.熟练使用Visual studio或Eclipse等常用编译设计软件。

9发展前景

行业发展前景：

现在,计算机和网络发展如火如荼,网络已经深入到每个人生活的各个方面。社会上IT行业对网络工程师的需求依旧不能被满足,尤其是大量的中小型企业急需有经验、有实力、有技能的网络工程师。因此,软件人才的需求比较强烈, 特别是web和app开发方面的人才。互联网行业总体来看是一个朝阳产业。

个人发展路径：

网络测试师—网络开发工程师—网络架构师—项目经理—技术总监

职业的就业趋势：

1.现在需求量比较大的研究方向是服务端设计、产品构架、html、 php、java、 mysql等前端和后端技术, 以及android和ios的客户端开发等。
2.行业内流动的机会比较多,一般都是从技术人员做起,后期发展有继续从事高级技术开发,也有转向管理岗位的。

10薪酬待遇

薪酬结构：

一般的薪酬结构:基础工资+岗位工资+年终奖

满意度：

4（薪酬满意度评分：1点~5点，分别代表“非常不满意”~“非常满意”）

福利制度：

可能有的福利制度:工龄工资、午餐补贴、交通补贴、社会保险(养老、医疗、失业、生育、工伤保险等)、住房公积金及国家法定节假日补贴等。

11职业道德

1.遵纪守法:每个从业人员都要遵守纪律和法律,尤其要遵守职业纪律和与职业活动相关的法律法规。
2.勤奋上进:以科学务实的态度、严谨细致的精神全身心投入工作。
3.保密精神:不能泄漏公司的数据和未公布的信息,不得单方邀请第三方加入项目团队。
4.团队意识:虚心,加强与同事的交流,与同事良好合作。
