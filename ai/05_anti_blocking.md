# 反爬策略

## 基础策略
- 使用稳定浏览器指纹与常规 UA
- 设置合理 timeout
- 页面切换后增加等待间隔
- 控制详情页抓取频率
- 失败请求使用重试而非瞬时高频重打

## Playwright 策略
- CHSI 动态列表页默认采用 Playwright
- 等待条件优先使用 selector + 页面文本 + loading 消失的组合判断
- 避免只依赖固定 sleep
- 必要时保存 screenshot 与 html 作为调试材料

## 风险信号
- 403
- 429
- 页面空白
- 表格未刷新
- 选择器失效

## 处理原则
- 降低频率
- 增加重试间隔
- 启用调试输出
- 保留 partial 文件
